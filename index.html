Portfolio

Flood analysis using Sentinel-1

This project mapped monthly flood patterns in Kasese District, Uganda, using Sentinel-1 radar and JRC Surface Water data. By masking permanent water and applying a -15 dB threshold, temporary floods were detected and summarized monthly. January showed the largest flood extent (28.9 km²). Built in Google Earth Engine, this workflow supports flood monitoring and disaster planning in vulnerable areas.

GPS tracker Animation

This animation visualizes a bike route recorded using GPS data captured in GPX format. The track was overlaid on a styled basemap and route layer to highlight the cyclist’s movement. Frames were processed and compiled using ImageMagick, creating a dynamic time-lapse that brings spatial activity to life through simple yet powerful command-line image manipulation.

Interactive Web map app

This web app displays points of interest in Nassau County, NY, using Google Apps Script and Leaflet.js. Users can filter businesses by category, city, or name, and view detailed popups with contact info, website links, and hours of operation—all visualized on an interactive map.

Spatial Database Management Systems

This project was developed for an events company to support the planning and execution of their annual festival. It utilized WebGIS to power a mobile-friendly interface that provided real-time spatial information to attendees and staff. The application allowed clients to locate key amenities such as stages, washrooms, and vendor areas, while also guiding performers to their assigned stages. By integrating geospatial tools into event logistics, the platform enhanced on-site navigation, improved service delivery, and streamlined operational workflows for organizers.
